# Binary Trees

Allocate and deallocate many binary trees of varying depth.

**Workload:** GC pressure, memory allocation/deallocation, tree traversal.

**Input:** Minimum depth (e.g., 21)

**Output:** Checksum of tree nodes at each depth level.

**Reference:** https://benchmarksgame-team.pages.debian.net/benchmarksgame/description/binarytrees.html
