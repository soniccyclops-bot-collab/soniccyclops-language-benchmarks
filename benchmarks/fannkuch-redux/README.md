# Fannkuch Redux

Count maximum flips needed to sort permutations of a sequence by reversing prefixes.

**Workload:** Integer computation, array manipulation, permutation generation.

**Input:** Sequence length (e.g., 12)

**Output:** Checksum and maximum flip count.

**Reference:** https://benchmarksgame-team.pages.debian.net/benchmarksgame/description/fannkuchredux.html
