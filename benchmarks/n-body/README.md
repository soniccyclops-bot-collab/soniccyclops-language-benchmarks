# N-Body

Model the orbits of Jovian planets using a symplectic integrator (leapfrog).

**Workload:** Heavy floating-point arithmetic, tight loops, struct/record access.

**Input:** Number of simulation steps (e.g., 50000000)

**Output:** Energy of the system before and after simulation.

**Reference:** https://benchmarksgame-team.pages.debian.net/benchmarksgame/description/nbody.html
